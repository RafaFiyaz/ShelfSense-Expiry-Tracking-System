# ShelfSense-Expiry-Tracking-System
Tracks product expiry and reduces wastage

## Problem
Many products go to waste because expiry dates are not tracked properly.

## Solution
ShelfSense helps users track product expiry and get alerts before items expire.

## Features
- Expiry tracking  
- Alerts for users  
- Simple and efficient system  

## Future Improvements
- Mobile app version  
- Smart notifications
